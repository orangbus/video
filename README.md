# How to use it?
## Web
```
git clone https://github.com/orangbus/video.git
cd video && yarn run build
cp -R dist public
```
wwwroot -> webPath to /video/public
## Local
```
git clone https://github.com/orangbus/video.git
yarn install 
# 调试
yarn serve
```
## Project setup
```
yarn install
```
### Lints and fixes files
```
yarn run lint
```
